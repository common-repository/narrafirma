Stop story collection

This is a simple page. Its purpose is to turn off all web-based story collections when you have stopped collecting stories.

What to do here

When you are sure you don't want to collect any more stories, click the button to stop web collection. If you need to re-enable web collection, go back to the Start story collection page.

Connections to other pages

The only impact this page has on your project is that participants can no longer contribute stories using a web-based survey. However, after you have stopped the web-based survey, you can still enter or import stories into a story collection.
