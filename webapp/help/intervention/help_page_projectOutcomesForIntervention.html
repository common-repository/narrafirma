Answer questions about project outcomes

On this page you can think about how the main part of your project (collection, possibly catalysis, and sensemaking) turned out for each participant group. Were people helped by what happened? If so, how were they helped? And if not, why not?

What to do here

For each participant group involved in your project, create a new set of outcomes. Answer the questions about each group.

Connections to other pages

Answering questions about project outcomes creates the recommendations you can see in the Design intervention plans page.
