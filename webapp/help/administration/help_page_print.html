Print

On this page you can print a project report: a record of everything you entered into the project, rendered as plain HTML. A project report does not include any stories, but it does include every choice you made and every word you typed while you were using NarraFirma to work on the project. A project report is a version of your project you can access after you stop using NarraFirma.

What to do here

Print a project report.

You can also use this page to follow links to print other things about the project.

Connections to other pages

This page depends on every other page in NarraFirma, but it has no impact on any other page.

Frequently-asked questions

Can I open a project report in my word processor?

Maybe. The HTML document uses CSS styles, and they are included in the file (instead of referencing an external CSS file). Some word processors can pick up CSS styles and use them as document styles. If that doesn't work, we recommend the use of Pandoc, which does a good job of converting HTML files to a variety of word processing formats, such as ODT and DOCX.

Can I print a project report, edit it, and then import it back into NarraFirma?

No. Unlike an exported JSON file, a printed HTML project report cannot be read by NarraFirma. It is for your use outside the program.
